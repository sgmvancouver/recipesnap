# /debug Command

Systematically debug and fix issues.

## Usage

```
/debug <issue>
```

## Examples

```
/debug Why login fails
/debug API returns 500 error
/debug Component not rendering
```

## Process

1. Reproduce the issue
2. Gather evidence (logs, errors)
3. Form hypotheses
4. Test hypotheses
5. Apply fix
6. Verify resolution
